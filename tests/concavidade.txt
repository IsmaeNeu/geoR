This data set contains measurements of soil properties

The columns corresponds to 
 1. X		: x-coordinate      
 2. Y       	: y-coordinate
 3. AMG     	: % de areia grossa
 4. AM          : % de areia media
 5. AF		: % de areia fina
 6. AMF		: % de areia muito fina
 7. Silte     	: % de silte
 8. Argila      : % de argila
 9. Altimetria  : cota do ponto
10. Superficie  : 1 - concava, 2 - convexa
